# 👑 Jesús, el Rey de Reyes

Aplicación web educativa cristiana para niños (3-10 años) con historias bíblicas, canciones, libros interactivos y juegos educativos.

## ✨ Características

- 📚 **Historias Bíblicas Interactivas**: Narraciones con ilustraciones atractivas
- 🎵 **Canciones para Aprender**: Letras con melodías suaves
- 📖 **Libros Interactivos**: Páginas con elementos táctiles
- 🎮 **Juegos Educativos**: Actividades lúdicas que refuerzan valores bíblicos
- 📱 **Diseño Mobile-First**: Experiencia óptima en teléfonos y tablets
- ♿ **Accesibilidad Total**: WCAG 2.1 AA
- ⚡ **Alto Rendimiento**: Lighthouse >90
- 📱 **PWA Listo**: Funciona offline

## 🚀 Inicio Rápido

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build
```
